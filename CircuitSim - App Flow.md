### **App Flow & User Journey**

**General User Flow**

* **Landing Page**

  * Overview, login/signup, feature highlights, pricing details.  
* **Authentication**

  * Signup/Login (social logins, email/password).  
  * Password Reset.  
* **Dashboard (Post Login)**

  * Create new project.  
  * View/manage existing projects.  
  * Access profile/account details.  
  * Subscription management (view plan, upgrade/downgrade).  
* **Project Editor**

  * Add components from library.  
  * Wire components interactively.  
  * Write/edit code for microcontrollers.  
  * Run simulation and view live updates.  
  * Access Serial Monitor for debugging.  
  * Version Control (history, rollback) – premium plan.  
  * Project Import/Export – premium plan.  
  * Real-time collaboration (invite users, manage concurrent edits) – premium plan.

**Admin Flow**

* **Admin Dashboard**  
  * User Management (create, edit, deactivate accounts, reset passwords).  
  * Plan Management (configure plan tiers/features).  
  * Component Library Management (add, edit, preview, remove components).  
  * App-wide Settings (logging, usage analytics).
